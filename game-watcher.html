About

Description

GameWatcher - Release Timer helps you to track the release dates of your favorite games. Let yourself be reminded regularly about the remaining time until the release. Or go a step further and personalize your home screen like you have never done before and show the remaining time until the release of your favorite game directly in a great widget. Get informed about changes to the release date of your favorite games.

Tech Details

Swift, Widgets, SwiftUI, CoreData, Java Spring Backend in AWS Elastic Beanstalk, Darkmode, Notifications, Networking, Custom Animations, Background Tasks

Game metadata is powered by IGDB.com
